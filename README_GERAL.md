# Organizador de Tarefas JWT

Sistema simples para atividade prática de testes.

Use o projeto `curadoria_jwt` como exemplo de solução pronta e peça aos alunos para implementarem uma suíte equivalente neste sistema.

Diferença principal:

- `curadoria_jwt`: exemplo com testes prontos.
- `organizador_tarefas_jwt`: exercício para os alunos criarem os testes.
